# Panel Escolar (PHP + MySQL)

## 1) Importar BD
- Abre phpMyAdmin
- Importar -> selecciona `database.sql`
- Esto crea la BD `panel_escolar` y agrega datos de prueba.

## 2) Colocar en XAMPP
- Copia la carpeta en `htdocs` (por ejemplo: `htdocs/panel_escolar_php`)
- Abre: `http://localhost/panel_escolar_php/index.php`

## 3) Cambiar credenciales
Si tu MySQL no es root sin password, edita: `includes/db.php`
